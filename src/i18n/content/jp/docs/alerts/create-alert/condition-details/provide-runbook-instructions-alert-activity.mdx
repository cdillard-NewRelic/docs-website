アラート [条件](/docs/alerts/new-relic-alerts-beta/configuring-alert-policies/define-alert-conditions) には、 [ランブック](/docs/using-new-relic/welcome-new-relic/get-started/glossary#alert-runbook) 指示のための URL を特定するオプションが含まれています。これにより、状況が条件の [クリティカル・スレッショルド](/docs/alerts/new-relic-alerts-beta/configuring-alert-policies/define-thresholds-trigger-alert) を超えたときに開かれるインシデントを処理するための情報や標準手順にリンクすることができます。

<figcaption>
  <DoNotTranslate>**[one.newrelic.com > All capabilities](https://one.newrelic.com/all-capabilities) > Alerts > Alert conditions (policies) > (select a policy) > (select a condition) > (edit the condition) > Add details > (scroll down to Runbook URL)**</DoNotTranslate>: 集計をトリガーしたインシデントを取り扱う担当者が対処方法を知ることができるように、ランブック URL を追加できます。
</figcaption>

条件に関連するランブックのURLを編集するには

1. <DoNotTranslate>**[one.newrelic.com > All capabilities](https://one.newrelic.com/all-capabilities) > Alerts > Alert conditions (policies) > (select a policy) > (select a condition)**</DoNotTranslate>

   に移動します。

2. <DoNotTranslate>**Edit conditions**</DoNotTranslate>

   ページで

   <DoNotTranslate>**Add details**</DoNotTranslate>

   をクリックし、

   <DoNotTranslate>**Runbook URL**</DoNotTranslate>

   まで下にスクロールします。 ランブック URL の完全なパス (ランブック URL、内部 Wiki ページへの URL など) を指定します。

3. <DoNotTranslate>**Enable on save**</DoNotTranslate>

   を切り替えて

   <DoNotTranslate>**Save**</DoNotTranslate>

   をクリックします。

4. オプション: アラート条件から既存のランブック URL を変更または削除するには、

   <DoNotTranslate>**Remove**</DoNotTranslate>

   をクリックします。

担当者が正確に何をすべきかわかるように、 <DoNotTranslate>**Incident**</DoNotTranslate>詳細と電子メール通知にランブックの URL を含めます。
