경고 [조건](/docs/alerts/new-relic-alerts-beta/configuring-alert-policies/define-alert-conditions) 에는 [Runbook](/docs/using-new-relic/welcome-new-relic/get-started/glossary#alert-runbook) 지침에 대한 URL을 식별하는 옵션이 포함됩니다. 이를 통해 상황이 조건의 [임계 임계값](/docs/alerts/new-relic-alerts-beta/configuring-alert-policies/define-thresholds-trigger-alert) 을 위반할 때 열린 사고를 처리하기 위한 정보 또는 표준 절차에 연결할 수 있습니다.

</figcaption>

조건과 연결된 Runbook URL을 편집하려면 다음을 수행합니다.

1. <DoNotTranslate>**[one.newrelic.com > All capabilities](https://one.newrelic.com/all-capabilities) > Alerts > Alert conditions (policies) > (select a policy) > (select a condition)**</DoNotTranslate>

   으)로 이동합니다.

2. <DoNotTranslate>**Edit conditions**</DoNotTranslate>

   페이지에서

   <DoNotTranslate>**Add details**</DoNotTranslate>

   클릭한 다음

   <DoNotTranslate>**Runbook URL**</DoNotTranslate>

   까지 아래로 스크롤합니다. 런북 URL의 전체 경로(런북 URL, 내부 위키 페이지 URL 등)를 제공하세요.

3. <DoNotTranslate>**Enable on save**</DoNotTranslate>

   전환하고

   <DoNotTranslate>**Save**</DoNotTranslate>

   클릭합니다.

4. 선택 사항: 공지 조건에서 기존 런북 URL을 변경하거나 삭제하려면

   <DoNotTranslate>**Remove**</DoNotTranslate>

   클릭하세요.

담당자가 무엇을 해야 할지 정확히 알 수 있도록 귀하의 런북 URL을 <DoNotTranslate>**Incident**</DoNotTranslate> 세부정보와 이메일 공지에 포함하겠습니다.
